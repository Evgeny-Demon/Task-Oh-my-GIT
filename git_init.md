__```git init```__
---
 Это как активировать режим ниндзя в своей папке.

Написал:
```bash= 
git init
```
и твоя папка стала репозиторием. Git теперь следит за каждым файлом, как Леонардо за твоими тренировками. Начни создавать файлы, потом [git add](./add.md) и [git commit](git_commit.md) и станешь кибер-ниндзя мастером версий.
