# __```git checkout```__ 
Команда __`git checkout`__ используется для переключения между ветками или восстановления файлов до определённого состояния.

Допустим Раф хочет переключиться на ветку *`training`*:
```bash=
git checkout training
```

Или создать и сразу перейти в новую ветку *`shredder-fix:`*
```bash=
git checkout -b shredder-fix
```

Восстановить файл __**dojo-plan.txt**__ до состояния из последнего коммита - отменить локальные изменения:
```bash=
git checkout HEAD -- dojo-plan.txt
```

Посмотреть, как выглядел файл __**nunchucks-config.js**__ три коммита назад:
```bash=
git checkout HEAD~3 -- nunchucks-config.js
```
У черепашек __`git checkout`__ — как телепорт в другое состояние проекта: то ли в прошлое, то ли в параллельную ветку миссии.
