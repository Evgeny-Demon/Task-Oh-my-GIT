# __```git fetch```__
 А это уже как тихо подсмотреть, что нового появилось в удалённом репозитории (например, у Рафаэля), не трогая свои файлы.

Он скачивает новые коммиты, ветки и теги — но ничего не сливает в твою рабочую ветку. Просто обновляет __"карту миссий"__ у себя в голове (локальные ссылки вроде __***origin/main***__).

Пример:
```bash=
git fetch upstream
```
Здесь ты узнаешь, что Рафаэль добавил новый рецепт пиццы, но твоя кухня пока не изменилась. Ты провел разведку.
