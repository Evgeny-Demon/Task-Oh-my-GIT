# __```git status```__
Команда __`git status`__ показывает текущее состояние рабочей директории: какие файлы изменены, добавлены в индекс, или не отслеживаются. Это как если бы Сплинтер заглянул в логово и спросил: «Что у вас тут происходит? Кто что правил?»

Майки только что создал файл nunchucks-guide.txt, но ещё ничего не делал с ним?
```bash=
git status
```
Git сообщит, что файл не отслеживается.

Донателло изменил *`drone-code.js`* и добавил его в коммит. Тогда:
```bash=
it add drone-code.js
git status
```
Git покажет, что файл готов к коммиту.

Или Леонардо изменил *`stealth-mode.cpp`*, но ещё не добавил в индекс:
```bash=
git status
```
Git укажет, что файл изменён, но не добавлен для коммита.
