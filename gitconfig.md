__```git config```__ 
---
Это как настроить свой профиль ниндзя в системе.
Хочешь, чтобы Git знал, кто коммитит? Скажи:

```bash=
git config --global user.name "Michelangelo"
git config --global user.email "mikey@tmnt.net"
```
`--global` — значит везде, на всех проектах». Без него — только в текущей папке.

Посмотреть настройки? Легко:

```bash=
git config --list
```
Если не настроишь — Git подумает, что за кодом скрывается Шредер. Оно нам надо?
