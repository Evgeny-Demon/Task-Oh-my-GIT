# __```git log```__
Команда __`git log`__ показывает историю коммитов в текущей ветке — кто что менял, когда и зачем. Это как хроника миссий черепашек: каждая запись — отчёт о выполненной работе.

Посмотреть полную историю коммитов:
```bash=
git log
```
Вывести краткую историю — по одной строке на коммит:
```bash=
git log --oneline
```
Увидеть, кто что менял и в каких файлах:
```bash=
git log --stat
```
Например, Леонардо может выполнить __`git log --oneline`__, чтобы быстро проверить, когда Донателло добавил обновления или когда Рафаэль в последний раз делал какие-то правки.  

Это основной инструмент для просмотра истории изменений — без изменений в коде, только информация.
