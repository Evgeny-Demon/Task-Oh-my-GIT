# __```git remote add```__ ***имя URL*** 
— просто запоминает адрес удалённого репозитория под коротким именем.
Например:  
```bash=
git remote add upstream https://github.com/leo/turtles-game.git
```

Теперь ты можешь легко получать обновления оттуда — без копирования, просто как «закладку» для Git.
