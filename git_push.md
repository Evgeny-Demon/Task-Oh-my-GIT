# __```git push```__
Команда __`git push`__ отправляет локальные коммиты в удалённый репозиторий — как будто Донателло закончил правку лазерной катаны и отправляет код на общий сервер, чтобы остальные черепашки могли его увидеть.

Предположим, Леонардо завершил улучшение системы уклонения и отправляет изменения в ветку __*`main:`*__
```bash=
git push origin main
```
или Донателло создал новую ветку *`drone-upgrades`* и впервые отправляет её на сервер:
```bash=
git push -u origin drone-upgrades
```
Микелянджело продолжает править ветку __*`pizza-recipes`*__ и просто обновляет её на удалённом репозитории:
```bash=
git push
```
