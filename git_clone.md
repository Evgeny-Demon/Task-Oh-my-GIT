# __```git clone```__
---
Представь, Майки: ты нашёл секретный свиток с боевыми техниками в храме GitHub — или, скажем, репозиторий с чертежами новой пиццерии. Но он там, в облаке. А тебе нужно скопировать его к себе в логово, чтобы тренироваться или вносить правки.

Для этого ты используешь:
```bash=
git clone https://github.com/leo/pizzeria.git
```

Команда `git clone` создаёт локальную копию удалённого репозитория — со всей его историей, ветками и коммитами. Она настраивает связь с оригинальным репозиторием (под именем ***origin***), чтобы потом можно было получать обновления или отправлять свои.
