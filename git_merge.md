# __```git merge```__
Это когда ты объединяешь чужие изменения (например, из ветки Леонардо) со своей работой.  

Допустим, ты в ветке `main`, а хочешь добавить обновления из `upstream/main:`
```bash=
git merge upstream/main
```
Git попытается автоматически *слить коммиты*. Если вы правили разные файлы — всё пройдёт гладко. Если одни и те же строки — будет конфликт, как будто Леонардо и Рафаэль ссорятся. И тогда тебе самому придётся решить, чья версия лучше.
