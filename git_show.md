# __```git show```__
Команда __`git show`__ выводит подробную информацию о конкретном коммите: кто его сделал, когда, какие изменения внесены, и сообщение коммита. Это как прочитать полный отчёт одной миссии черепашек, а не просто её название.

Посмотреть самый последний коммит:
```bash=
git show
```

Посмотреть конкретный коммит по хешу (первые 7 символов обычно достаточно):
```bash=
git show a1b2c3d
```

Посмотреть, что изменил Донателло в своём последнем коммите, даже если он не последний в истории:
```
bash=
git show HEAD~2    # показать коммит два шага назад от текущего
```

Показать только изменения в определённом файле в рамках коммита:
```bash=
git show a1b2c3d:splinter-inventory.txt
```
__`git show`__ — это детальный взгляд на один конкретный момент истории, в отличие от [__`git log`__](/git_log.md), который показывает обзор всей ленты событий.
